## OpenType Features to Enable

### Crimson Pro (Display)
```css
font-feature-settings:
  'liga' 1,   /* Ligatures: ff, fi, fl */
  'kern' 1,   /* Kerning: AV, WA */
  'ss01' 1;   /* Sharper serifs (more crystalline) */

letter-spacing: -0.02em; /* Tighter for elegance */
```

### Work Sans (Body)
```css
font-feature-settings:
  'liga' 1,   /* Standard ligatures */
  'kern' 1,   /* Kerning */
  'tnum' 1;   /* Tabular numbers (data tables) */

letter-spacing: -0.01em; /* Slightly condensed */
```

### JetBrains Mono (Code)
```css
font-feature-settings:
  'liga' 1,   /* Code ligatures: !=, =>, >= */
  'calt' 1,   /* Contextual alternates */
  'zero' 1;   /* Slashed zero: 0 vs O */
```
