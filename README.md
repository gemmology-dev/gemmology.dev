# gemmology.dev

Main website for the [Gemmology Project](https://gemmology.dev).

## Features

- CDL Playground - Interactive crystal description editor
- Crystal Gallery - Browse 46+ mineral presets
- Documentation hub
- FGA curriculum content

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed to Cloudflare Pages on push to main.
